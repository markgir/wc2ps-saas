Processo

Simples como devia ser

Sem ferramentas complicadas. Sem exportar CSVs. Sem perder dados. A migração corre directamente entre as duas bases de dados.

01

Escolhes o plano

02

Instalar o agent

Carregas um único ficheiro PHP para a raiz do teu hosting via File Manager. Sem SSH, sem linha de comandos.

03

Introduzes as credenciais das bases de dados WooCommerce e PrestaShop. A ligação é testada antes de começar.

04

Migração autónoma

O servidor processa tudo em background. Podes fechar o browser — a migração continua. Recebes email no final.

Perguntas frequentes

Tens dúvidas?

Preciso de acesso SSH ao servidor?

Não. Apenas fazes upload de um ficheiro PHP via File Manager do cPanel ou ISPmanager. Sem linha de comandos, sem configurações técnicas complexas.

O WooCommerce e o PrestaShop têm de estar no mesmo servidor?

Não. Podem estar em servidores diferentes. O agent instala-se em ambos e comunica com o nosso servidor de processamento via HTTPS.

O que acontece se a migração falhar a meio?

A migração guarda o estado a cada batch. Podes retomar exactamente onde parou sem repetir o que já foi migrado. Nenhum dado é perdido.

As minhas credenciais ficam guardadas nos vossos servidores?

As credenciais da base de dados nunca saem do teu servidor. O nosso servidor comunica com o agent via token — nunca vê as tuas passwords.
